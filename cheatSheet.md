# 基本

サーバ起動  
hexo server
hexo server --draft  
hexo generate
hexo deploy

ドラフト作成  
hexo new draft {draft名}  
hexo publish {draft名}    

諸々きれいにする  
hexo clean

まとめてやる
hexo clean && hexo generate && hexo deploy
